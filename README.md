# Computational Investigation of Dynamic Stability Derivatives for Semi-Ballistic Re-entry Modules

## Overview
This project investigates the dynamic stability characteristics of semi-ballistic re-entry modules through computational methods.

The study focuses on determining the dynamic stability derivatives that influence the aerodynamic behaviour of re-entry vehicles during atmospheric descent.

## Objectives
- Develop a CAD model of a semi-ballistic re-entry module using PTC Creo
- Analyze aerodynamic characteristics
- Compute dynamic stability derivatives
- Study the stability behaviour during re-entry conditions

## Tools & Software
- PTC Creo (CAD modelling)
- Python
- NumPy
- Matplotlib

## Methodology
1. Literature review of re-entry vehicle aerodynamics
2. Geometry creation of re-entry module
3. Computational aerodynamic analysis
4. Determination of dynamic stability derivatives
5. Visualization and interpretation of results

## Repository Structure
- `cad_model` – CAD geometry of re-entry module
- `analysis` – Python scripts for stability analysis
- `results` – Generated graphs and data
- `docs` – Project report and literature review
- `images` – CAD screenshots and geometry visuals
